# Project Atlas

Project Atlas is a self-hosted infrastructure and AI development environment designed for learning and deploying real-world DevOps, Linux, networking, and cybersecurity systems.

## Goals
- Build a reproducible homelab environment
- Learn Linux system administration
- Learn Docker and container orchestration
- Build a self-hosted AI development platform
- Practice cybersecurity and network engineering skills
- Create portable infrastructure deployable across multiple machines

## Current Architecture
- Laptop: development and testing node
- Main PC (3080): AI compute and heavy workloads
- Proxmox server: stable services (media, automation)
- Future: distributed mini PC cluster

## Principles
- Everything must be reproducible
- Everything is version controlled
- Everything is documented
- Prefer containers over native installs
- Assume systems will fail and must be recovered
