# Project Atlas Learning Journal

## Day 1 - Foundation Setup

### What I’ve done so far
- Set up Git and GitHub with SSH authentication
- Learned the difference between HTTPS and SSH authentication
- Installed and fixed Docker on Linux
- Ran my first containers
- Built a basic Docker Compose stack (nginx)
- Learned container lifecycle (up / down)

### Key concepts learned
- Docker daemon must be running for containers to work
- Containers are ephemeral unless configured otherwise
- Docker Compose defines infrastructure as code
- Linux permissions affect Docker access (docker group)
